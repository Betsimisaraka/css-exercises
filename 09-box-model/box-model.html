Box model

Box model is a box which is the content surrounded by . Every html have their own box model. The box model is including margin, border, padding, and content. Creating a box model is deppent on how many values you have put with these properties. It is better to specify the width and the height of the box model. Understanding these boxes will help you to be able to create a layout of your document and make it easily work on. The box model is also used to decorate the content and it can affect how it looks inside the box.
